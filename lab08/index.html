Form Elements

An overview of five common form elements and their uses.

Text Box

Text box fields allow users to type in text or numeric information such as names, e-mail addresses, phone numbers, and other text.

Checkbox

The check box form control allows the user to select one or more options from a group of items.



Radio Buttons

Radio buttons allow users to select only one option from a list.



Submit Button

The submit button allows users to submit the form.

Reset Button

The reset button allows users to reset the form.
