Introduction to XML and JavaScript

What is XML?

XML (Extensible Markup Language) is a markup language and file format for storing and transporting data.

How is XML used?

XML is used to exchange, organize, and seperate data from presentation.

What does XML look like? (Example)

        <song>
            <title>Bullet Proof Soul</title>
            <artist>Sade</artist>
            <album>Love Deluxe</album>
            <year>1992</year>
            <duration>5:25</duration>
        </song>

What is JavaScript?

JavaScript is an object-based, client-side scripting language that allows developers to make web pages dynamic and interactive.

How is JavaScript related to HTML and CSS?

HTML (Hypertext Markup Language) structures the content on web pages, CSS (Cascading Style Sheets) styles the appearance of HTML elements, and JavaScript can update and change both HTML and CSS. JavaScript also adds interactivity and dynamic behavior to web pages.

Common JavaScript Elements, Properties, Methods, and Functions (Example)

  • Element: document.getElementById('example')

    Selects an HTML element with the specified id attribute.

  • Property: element.style

    Represents an element's inline CSS styles, which allow you to directly change how it looks.

  • Method: element.addEventListener('click', function(){ /* do something */ })

    Attaches an event handler function to an HTML element and identifies the type of event and the function to execute when the event occurs.

  • Function: function myFunction() { /* do something */ }

    Defines a custom function named "myFunction" that performs a specific task.
